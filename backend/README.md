# Ejemplo Pruebas Unitarias y de Integración}

## V1.0.0
En esta versión se implementan pruebas unitarias para el modelo de datos y pruebas de integración para las rutas de la API. Se utilizan herramientas como Jest para las pruebas unitarias y Supertest para las pruebas de integración. Se incluyen casos de prueba para validar la lógica del modelo y la funcionalidad de las rutas, asegurando que el backend funcione correctamente y maneje los casos de error adecuadamente.
